# Highlights

## Run Locally

Clone the project

```bash
  git clone https://github.com/Bhargavi952/tericsoft-assignment.git
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm  start
```


### Deployed link:
- https://tericsoft-highlights-application.netlify.app/


### Heroku mock server:
